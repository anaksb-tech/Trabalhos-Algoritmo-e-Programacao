# Trabalhos de Algoritmos e Estruturas de Programação

Este repositório armazena as soluções para os trabalhos propostos durante a disciplina de Algoritmos e Programação (AEP). As implementações estão em C e os arquivos contêm o código-fonte (.c) de cada solução.

## Conteúdo do Repositório

Os trabalhos estão organizados por entregas. Cada pasta contém os arquivos fonte e um README explicando o que cada programa faz.

**Trabalho 1 - Manipulação Numérica e Orçamento:**
Programas desenvolvidos em C com foco em operações sobre números inteiros e cálculos financeiros. O primeiro programa realiza rotações e inserções de dígitos sobre um número lido pelo usuário. O segundo calcula orçamentos de instalação de ar-condicionado com base em mão de obra, tipo de ambiente, estação do ano e forma de pagamento.

**Trabalho 2 - Busca em Matriz e Sistema de Agenda:**
Programas desenvolvidos em C com foco em manipulação de matrizes e gerenciamento de dados com structs. O primeiro programa busca uma sequência de números em uma matriz nos quatro sentidos possíveis. O segundo implementa um sistema completo de agenda para um consultório dentário, com cadastro de pacientes, agendamento de consultas e listagens ordenadas.
